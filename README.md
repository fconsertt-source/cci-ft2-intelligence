# ft2‑reporting

FT2 data processing and reporting pipeline.

## ما هو FT2؟

FT2 هو تنسيق بيانات يُستَخدم في نظام التبريد اللقاح (Cold‑Chain).  
هذا المستودع يقدّم:

* قراءة وتحليل ملفات FT2
* توليد تقارير PDF موحّدة
* واجهة سطر أوامر موحدة (`ft2-cli`) مبنية على **Typer**

## تشغيل المشروع

```bash
# تثبيت الاعتمادات
poetry install

# عرض دليل الأوامر
poetry run ft2-cli --help
```

> **ملاحظة:** المشروع يستخدم **Clean Architecture**؛ الطبقات `application`, `domain`, `infrastructure` مفصولة بوضوح.

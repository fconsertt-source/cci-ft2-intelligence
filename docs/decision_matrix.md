# مصفوفة اتخاذ القرار (Decision Matrix) - نظام CCI-FT2

## 1. نظرة عامة
يعتمد النظام على معايير منظمة الصحة العالمية (WHO) ومراكز السيطرة على الأمراض (CDC) لتقييم سلامة اللقاحات في سلسلة التبريد. يتم اتخاذ القرارات بناءً على تحليل البيانات الخام من أجهزة FT2 وتطبيق قواعد صارمة لضمان سلامة اللقاحات.

## 2. القواعد الأساسية (Core Rules)

### أ. قاعدة التجميد (Freeze Rule - Zero Tolerance)
*   **الشرط:** أي انخفاض في درجة الحرارة تحت `0.0°C`.
*   **القرار:** `REJECTED_FREEZE`
*   **التفسير:** اللقاحات الحساسة للتجميد (مثل الخماسي، IPV) تتلف فورياً عند التجمد. لا يوجد تسامح مع هذه الحالة.
*   **الاستثناء:** إذا تم تحديد المركز كـ `freeze_sensitive = False` (يحتوي فقط على لقاحات مقاومة مثل OPV)، يتم تسجيل الواقعة ولكن لا يتم رفض الشحنة فورياً بناءً على التجميد فقط.

### ب. قاعدة الحرارة الحرجة (Critical Heat Rule)
*   **الشرط 1:** درجة الحرارة القصوى `> 10.0°C`.
*   **الشرط 2:** تجاوز الحد التراكمي للحرارة (CCM Limit) - افتراضياً 600 دقيقة فوق 8°C.
*   **القرار:** `REJECTED_HEAT_C`
*   **التفسير:** التعرض لدرجات حرارة عالية جداً أو لفترات طويلة يؤدي إلى تدهور فعالية اللقاح بشكل لا رجعة فيه (المرحلة C/D من VVM).

### ج. قاعدة التحذير (Warning Rule)
*   **الشرط:** درجة الحرارة بين `0.0°C - 2.0°C` أو `8.0°C - 10.0°C`.
*   **القرار:** `ACCEPTED` (مع علامة `WARNING_EXCURSION`)
*   **التفسير:** خروج طفيف عن النطاق المثالي (2-8°C). لا يستدعي الإتلاف الفوري ولكن يتطلب مراجعة وفحص VVM وتسجيل الواقعة.

### د. الحالة الطبيعية (Normal State)
*   **الشرط:** درجة الحرارة ضمن النطاق `2.0°C - 8.0°C`.
*   **القرار:** `ACCEPTED`

## 3. جدول القرارات والإجراءات

| القرار | الحالة | الإجراء الموصى به |
| :--- | :--- | :--- |
| **REJECTED_FREEZE** | تجميد (< 0°C) | إتلاف اللقاحات الحساسة للتجميد فوراً. فحص VVM للمقاومة. |
| **REJECTED_HEAT_C** | حرارة حرجة (> 10°C أو تراكمي) | إتلاف أو فحص دقيق (Action Required). |
| **WARNING_EXCURSION** | تحذير (0-2°C أو 8-10°C) | مراقبة دقيقة، تسجيل في السجلات، وفحص VVM. |
| **ACCEPTED** | سليم (2-8°C) | اللقاح آمن تماماً للاستخدام المباشر. |
| **NO_DATA** | لا توجد بيانات | التحقق من سلامة جهاز المراقبة. |

## 4. أولويات التنفيذ (Execution Priority)
يتم تطبيق القواعد بالتسلسل التالي (الأول يغلب):
1.  **التجميد:** له الأولوية القصوى (إتلاف فوري للحساس).
2.  **الحرارة الحرجة:** تأتي ثانياً (إتلاف بسبب الحرارة).
3.  **التحذير:** يتم تسجيله دون تغيير قرار القبول ما لم توجد مخالفات أخرى.
4.  **القبول:** الحالة الافتراضية.
